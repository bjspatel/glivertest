***** Steps for Installation *****

1) Unzip glivertest-master.zip file
2) Download and Install MySQL
3) Create a database named "group_chat"
4) Download and Install Node.js
5) Open command prompt and cd to the unzipped folder "glivertest-master"
6) Run command: "npm install"

***** Steps to run server *****
1) Open command prompt and cd to the unzipped folder "glivertest-master"
2) Run command: "node server.js"

***** Steps to use application *****
1) Open browser
2) Open "http://<ip_address_of_server>:3000" (if server is running on local machine, then you can use: http://localhost:3000)
